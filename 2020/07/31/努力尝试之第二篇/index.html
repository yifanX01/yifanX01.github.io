如何上传文件

  1. sudo su 打开管理员权限
  2. cd blog 进入blog （pwd可检查位置）
  3. hexo new xxx 新建文档（文档必须是.md，放在sources/posts里）
  4. hexo clean
  5. hexo g
  6. hexo s 去链接检查一下
  7. Crtl C
  8. hexo d 上传
  • hexo clear 解释代码
